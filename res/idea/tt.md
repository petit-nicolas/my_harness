## Atelier de création vectorielle SVG
Générer et itérer des graphiques SVG par le biais d'un dialogue — logos, icônes, illustrations, infographies.

Pourquoi cela ne peut-il pas être accompli par une simple fonctionnalité (skill) ? Une fonctionnalité ne permet qu'à un modèle de générer du code SVG, et vous devez ensuite copier-coller ce code dans un navigateur pour visualiser le résultat. En revanche, un Agent vertical dispose de :

- Un panneau de rendu SVG en temps réel sur la gauche, où chaque modification de code est immédiatement visible
- L'outil « shape_library » — une bibliothèque préconfigurée de plus de 200 formes et motifs de base, que le modèle peut combiner plutôt que dessiner à partir de zéro
- Le gestionnaire de calques (« layer_manager ») — une gestion des calques permettant de modifier individuellement chaque calque via un dialogue
- L'outil « export_variants » — exportation simultanée dans plusieurs formats et tailles (favicon, icône d'application, image de profil pour les réseaux sociaux)
- Un historique persistant des versions de conception — possibilité de revenir sur n'importe quelle itération précédente

Valeur ajoutée : Les développeurs indépendants ou les créateurs de contenu sur les réseaux sociaux ont souvent besoin de graphiques vectoriels simples, sans vouloir ouvrir Figma ou Illustrator. Une simple instruction telle que « Créez-moi une icône de tasse à café dans un style minimaliste, avec une épaisseur de trait de 2 px et des teintes chaudes » suffit pour obtenir l'image et effectuer des itérations.
